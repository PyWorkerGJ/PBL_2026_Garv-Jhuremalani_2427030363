Plant Disease Detector with AI Image Validation System

Two-stage deep learning system using ResNet-18 that verifies authenticity before disease classification, ensuring reliable plant disease diagnosis.

Problem Statement

AI-generated images can mislead disease classifiers. Existing systems assume authenticity.

Objectives

  • Detect fake plant images
  • Classify diseases accurately
  • Ensure reliable AI predictions

Methodology

PlantVillage dataset + synthetic fake images

ResNet-18 CNN models

Two-stage secure pipeline

Literature Review

Mohanty et al. achieved 99% accuracy using PlantVillage dataset but ignored image authenticity.

Ferentinos compared CNNs but lacked fake image detection.

Enhancements

Webcam capture via Streamlit for real-time detection.

Recommendation engine for treatment guidance.

Results

Accuracy: 96.4%

Measured using unseen test dataset.
