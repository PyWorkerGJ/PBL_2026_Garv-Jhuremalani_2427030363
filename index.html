Plant Disease Detector with AI Image Validation System

Two-stage deep learning system using ResNet-18 that verifies authenticity before disease classification, ensuring reliable plant disease diagnosis and preventing AI-generated image misuse.

Problem Statement

Traditional plant disease detection assumes all images are authentic. AI-generated images can mislead classifiers. This system introduces authenticity verification before classification to ensure reliable predictions.

Objectives

  • Detect fake or AI-generated plant images
  • Classify diseases accurately using CNN
  • Build secure two-stage AI pipeline
  • Improve reliability of AI predictions

Methodology

Dataset

PlantVillage used for real images. Fake images generated using Stable Diffusion and augmentation.

Model

ResNet-18 CNN trained using PyTorch for authenticity and disease classification.

Pipeline

Image → Fake Detector → Disease Classifier → Recommendation → Output

Literature Review

Mohanty et al. (2016)

CNN achieved >99% accuracy using PlantVillage dataset but lacked authenticity verification.

Ferentinos (2018)

Compared deep learning architectures but did not address fake image detection.

Research Gap

Existing systems focus only on classification accuracy and ignore authenticity verification. This project solves that limitation.

Enhancements

Webcam Integration

Streamlit camera_input enables real-time leaf capture and instant prediction without uploading files.

Recommendation Engine

Maps detected disease to treatment advice, transforming system into decision-support tool.

Results

Accuracy: 96.4%

Calculated using test dataset evaluation. Fake image detector and disease classifier together provide reliable predictions.
